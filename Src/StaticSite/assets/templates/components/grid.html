Grid system

The grid is a 12 column grid created with susy. We use the grid in two different ways ($grid-outer) to create the grid. It's a fluid grid with a fixed padding as gutter. How many columns an element should span over is set in the css there are 4 classes that is used to compose the column layout for pages. These are:

  • .container--12 - to span over the whole content width
  • .container--6 - to span over half the content width
  • .container--4 - to span over a third of the content width
To create columns inside a component the inside grid is used. It's also a fluid grid with the gutter as a fixed padding. Set how many columns you want the content to span over with: 
.content-parent {
    @include layout($grid-inner);
    @include container();
}
.content-child {
    @include span(12);
}
